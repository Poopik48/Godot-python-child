# Planet Jumper 🌌

3D платформер, где ты прыгаешь между маленькими планетами, собираешь монетки и стреляешь в летающих тварей. Каждая планета уникальна — её форма создаётся процедурно с помощью шейдеров.

---

## 🎮 Как играть

- **W/A/S/D** — движение  
- **Space** — прыжок  
- **ЛКМ** — атака  
- **Shift** — ускорение  

Цель: прыгай с планеты на планету, собирай монеты и не дай врагам догнать тебя.

---

## 🌍 Игровой процесс

- **Процедурные планеты** — каждая своя, со случайной формой и цветом  
- **Летающие враги** — атакуют, но их можно победить  
- **Монетки** — собирай для очков  
- **Гравитация** — притягивает к центру планеты, можно прыгать даже по стенкам
